# Craft Beer Store - Django E-Commerce Application

A modern, beautifully designed e-commerce web application for a craft beer store built with Django, Django REST Framework, and JWT authentication.

## Features

- 🍺 **6 Craft Beer Products** with detailed information
- 🎨 **Modern, Responsive Design** with beautiful UI/UX
- 🔐 **JWT Authentication** with Django Simple JWT
- 🚀 **RESTful API** with Django REST Framework
- ⚡ **Rate Limiting** for API security
- 📱 **Mobile-First Design** with responsive layouts
- 🛒 **Shopping Cart Functionality** (UI ready)
- 📝 **Contact Form** with email notifications
- 🔍 **Product Filtering** by category, style, and search
- ⭐ **Featured Products** section
- 🎯 **SOLID Principles** applied throughout
- 📊 **Admin Dashboard** for product management

## Technology Stack

- Django 4.2.7
- Django REST Framework 3.14.0
- Django Simple JWT 5.3.0
- Django Ratelimit 4.1.0
- Django CORS Headers 4.3.1
- WhiteNoise 6.6.0
- Pillow 10.1.0
